# AR-Object-Finder
**프로젝트 요약**: AR 마커를 활용한 물건 관리 및 공간 구조화 애플리케이션
개발 환경 및 기술 스택: Unity2022, C#, AR
플랫폼: Mobile



## 핵심 기능
### 1. AR 마커 배치
**물건 위치 등록:** 특정 물건이나 구조물에 AR 마커를 표시하여 위치를 직관적으로 파악할 수 있게 도와줍니다.
**구조도(Blueprint) 생성:** 마커의 형태를 변형해서 물건이 아니라 벽이나 기둥을 표시한다면 
방, 건물 등의 구조를 시각화 할 수 있습니다.

### 2. 효율적인 마커 검색 시스템
**대상 검색:** 찾고자 하는 물건을 이름으로 검색할 수 있습니다.
화면엔 검색된 마커만 출력(활성화)되서 쉽게 식별할 수 있니다. 나머지 마커들은 화면에 노출되지 않습니다(비활성화).

### 3. 카테고리 필터링 기능
**분류:** 마커별로 카테고리를 지정하여 분류할 수 있습니다.
**필터링:** 유저가 카테고리 활성화/비활성화 설정을 할 수 있습니다. 화면에는 활성화된 카테고리의 마커들만 노출됩니다.



## 스크립트 설명
### 핵심 스크립트
PointManager : 마커 제어 기능들을 인터페이스로 제공한다.

### PointManager가 제공하는 인터페이스를 이용하는 창(UI) 관리자 스크립트들
InfoPanelManager : 마커 정보 확인창 관리자.
CategorySettingManager : 카테고리 설정창 관리자.
MainButtonsManager : 기본창 관리자.
SelectButtonsManager : 마커 선택창 관리자.
EditDiagramManager : 마커 도형 수정창 관리자.
SearchManager : 마커 탐색창 관리 스크립트.

### 마커 정보 저장
PointController : 마커 정보 저장 및 마커 이름표 회전 스크립트. 마커 객체마다 존재한다. 
이름표가 늘 사용자를 바라보게 만든다(빌보드 효과)

### 기타 씬(Scene)에서 사용되는 스크립트들
ButtonManager : 공간 선택씬 제어 스크립트.
Smanager : 씬 전환기능만 제공하는 스크립트.
